# FUTURE_FS_03

## 🍰 **FreshBite Cafe Website**

A modern and responsive **Cafe & Restaurant Website** built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

This project showcases a professional restaurant website layout with menu browsing, gallery, reservations, and contact features designed to improve a cafe’s online presence.

## 📌 **Project Overview**

The **FreshBite Cafe Website** is a multi-page frontend web application created to demonstrate modern UI design and responsive web development.

The website allows visitors to:

✔ **Explore the cafe menu**
✔ **View food and cafe gallery images**
✔ **Learn about the cafe**
✔ **Book a table reservation**
✔ **Contact the cafe**

The project focuses on:

- **Clean UI design**
- **Responsive layout**
- **Interactive user experience**
- **Smooth navigation**
- **Modern web development practices**

## ✨ **Key Features**

### 🏠 **Homepage**

The homepage introduces the cafe with a visually appealing layout.

**Features include:**

Hero banner section
- **Featured dishes**
- **Special offers**
- **Call-to-action buttons**

### 🍽️ **Menu Section**

The menu page displays available food and drinks.

**Features include:**
- **Menu categories**
- **Food item descriptions**
- **Price display**
- **Order Now option**

Users can easily browse items and select dishes.

### 🖼️ **Gallery**

The gallery page showcases the cafe environment and food images.

**Features include:**
- **Image grid layout**
- **Lightbox image preview**
- **Smooth hover effects**

### 📅 **Reservation System**

Customers can reserve a table using the reservation form.

**Features include:**
- **Customer name input**
- **Date and time selection**
- **Number of guests**
- **Form validation**

### 📞 **Contact Page**

Allows customers to contact the cafe.

**Features include:**
- **Contact form**
- **Email and phone information**
- **Location details**

### 🎨 **UI & User Experience**

The website includes several modern UI features:

✔ **Sticky navigation bar**
✔ **Fully responsive design** (mobile / tablet / desktop)
✔ **Smooth scrolling**
✔ **Scroll progress bar**
✔ **Back-to-top button**
✔ **Animated hover effects**
✔ **Modern typography** using Google Fonts

## 🚀 **How to Run the Project**

### 1️⃣ **Clone the Repository**
```bash
git clone https://github.com/yourusername/freshbite-cafe-website.git
cd freshbite-cafe-website
```

### 2️⃣ **Open the Website**

Open **index.html** in your browser.

### 3️⃣ **Recommended Method (VS Code)**

Use **Live Server extension**.

**Steps:**
- **Install Live Server**
- **Right-click index.html**
- **Click Open with Live Server**
